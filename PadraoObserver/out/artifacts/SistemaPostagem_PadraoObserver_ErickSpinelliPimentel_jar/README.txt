baixar o .jar chamado SistemaPostagem_PadraoObserver_ErickSpinelliPimentel
e executar com o comando java -jar SistemaPostagem_PadraoObserver_ErickSpinelliPimentel.jar

COMANDO 1 (adicionar) :
add email
ex: add erick@gmail.com

COMANDO 2 (deletar):
del email
ex: del erick@gmail.com

COMANDO 3 (enviar mensagem):
email "mensagem"
ex: erick@gmail.com "ola galerinha"

COMANDO 4 (finalizar):
finalizar
ex: finalizar
